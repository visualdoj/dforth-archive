========================================
Использовать команды отсюда
========================================

m4 c_{action}.m4 {input}.cmd > output

  Применить преобразование action к файлу input.cmd.
  В файле input.cmd должны присутствовать описания команд.

m4 g_action.m4 groups.list > output

  Применить преобразование action к файлу со списком
  файлов из DEmbro\commands\*.cmd.
