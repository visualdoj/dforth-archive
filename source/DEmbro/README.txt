========================================
Как компилировать
========================================

1. Необходимо настроить unix utils и директории
   Для windows этот процесс автоматизирован батником
     dembro\trunk\release\utils\windows\makepath.bat
   Для unix настройка unix utils не нужна :)
   а скрипт для директорий будет в скором времени написан

2. Установить fpc, добавить пути до его бинарников в PATH

3. См. README
